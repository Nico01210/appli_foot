Règles du jeu

Découvrez comment jouer et gagner des points

Comment jouer

  • Sélectionnez un match : Cliquez sur "Pronostiquer" pour un match à venir
  • Saisissez votre pronostic : Entrez le score que vous prédisez (ex: 2-1)
  • Validez avant le coup d'envoi : Les pronostics ne sont plus acceptés une fois le match commencé
  • Suivez vos résultats : Consultez vos points dans le classement

Système de points

+5 points
Score exact : Vous avez prédit le score final exact (ex: 2-1 → 2-1)
+3 points
Résultat correct : Vous avez trouvé le bon vainqueur (ex: victoire prédite → victoire réelle)
+1 point
Vainqueur après temps réglementaire (phases finales) : Vous avez trouvé le bon vainqueur en Coupe du Monde
0 point
Pronostic incorrect : Vous n'avez pas trouvé le bon résultat

Délais et contraintes

  • Deadline : Les pronostics doivent être saisis avant le début du match
  • Modification : Vous pouvez modifier votre pronostic tant que le match n'a pas commencé
  • Décompte : Un timer vous indique le temps restant pour pronostiquer
  • Validation : Les résultats sont calculés automatiquement après chaque match

Classement

  • Classement général : Cumul de tous vos points depuis le début
  • Classement mensuel : Points gagnés ce mois-ci uniquement
  • Classement hebdomadaire : Points gagnés cette semaine
  • Mise à jour : Le classement est actualisé en temps réel
